# 00: Environment, IDEs, and CLI

## 1. VS Code and IDEs
- **IDE (Integrated Development Environment):** A software suite that consolidates basic tools required to write and test software. VS Code is a powerful text editor equipped with extensions, a file explorer, and an integrated terminal.
- **Why it matters:** Unlike simple text editors (like Notepad), IDEs provide syntax highlighting, auto-completion, and direct execution capabilities, drastically accelerating development speed.

## 2. Command-Line Interface (CLI)
- A text-based interface used to interact with the operating system and run programs.
- Instead of clicking buttons on a GUI (Graphical User Interface), developers type explicit commands (e.g., `cd`, `ls`, `mkdir`) to navigate and control the computer. This is faster, scriptable, and consumes fewer resources.

## 3. The Python Interpreter
- Python is an **Interpreted Language**. When you run a command like `python hello.py` in the CLI, the **Python Interpreter** reads your source code top-to-bottom, line-by-line, and translates it into machine code on the fly.
- **Compiled vs Interpreted:** Unlike C or C++ (which must be compiled entirely into a binary `.exe` file before running), Python executes dynamically, making it slower but highly portable across different operating systems.
